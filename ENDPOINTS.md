# Endpoints
This document describes all endpoints that need to be implemented.

## User Endpoints
**GetUserByID**
| Returns a User

**CreateUser**
| Returns the new User

## Contact Endpoints
**GetAllContactsByID**
| Returns all Contacts of the user (User ID's)

**CreateContact**
| Returns the new Contact

## Infected Endpoints
**GetInfectedByID**
| Returns the Infected

**CreateInfected**
| Returns the new Infected
